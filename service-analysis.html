Scientific Programming & Automation

Developing custom software solutions for complex measurement systems and data analysis workflows. My programming expertise enables efficient automation and reliable data processing.

  • Python Development: Comprehensive programming for measurement control, data acquisition, and analysis with libraries like NumPy, SciPy, and Matplotlib.
  • Measurement Automation: Custom control systems for four-point probe measurements, temperature monitoring, and voltage data acquisition.
  • Real-time Processing: Live data visualization and analysis during experiments for immediate feedback and quality control.
  • Cross-platform Solutions: Software that works across different operating systems and hardware configurations.
Development Environment & Workflow

Efficiency and collaboration are key in modern scientific programming. I primarily utilize Visual Studio Code for its versatility and extensive ecosystem, enhanced with GitHub Copilot for AI-assisted development while maintaining scientific rigor.

My experience extends beyond one environment. I am comfortable working with Arduino IDE for hardware interfacing and microcontroller programming in measurement systems, and Jupyter Notebooks for interactive data analysis and research documentation.

Effective version control is crucial for managing complex research projects and ensuring reproducibility. I am proficient with Git for tracking changes and managing code versions, and regularly use GitHub for repository hosting, collaboration, and maintaining research project documentation.
